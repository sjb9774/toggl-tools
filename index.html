toggl-tools

toggl-tools is a command line tool for using toggl that focuses on simplicity, utility, and productivity.

The workflow is simple and unobstrusive

user@computer(master)~$ toggl-setup this --project "Research"

user@computer(master)~$ toggl start this

Tracking "master" entry.

user@computer(master)~$ git checkout other_branch

user@computer(other_branch)~$ toggl stop

Stopped tracking "master".

user@computer(other_branch)~$ toggl start this

Tracking "other-branch" entry.
